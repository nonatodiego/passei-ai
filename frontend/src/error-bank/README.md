# Error Bank Module

Registra erros como fatos estruturados para Review, Recommendation, Analytics e Today. A interface nao cria prioridades; ela mostra o contexto, acao corretiva e reforco futuro definidos por contratos.

Inclui cadastro manual acessivel, validacao tipada, filtros, KPIs, lista, drawer de detalhes, acoes mockadas com feedback e estados loading, empty, error, success e todos dominados.
