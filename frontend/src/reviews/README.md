# Reviews Module

## Objetivo

Gerenciar revisoes e recorrencia de conteudos.

## Responsabilidades

- Revisoes pendentes.
- Revisoes concluidas.
- Regras de exibicao de revisao.
- Integracao futura com revisao espacada.

## Fluxo

O modulo apresenta revisoes recomendadas pelo Study Engine ou vindas de services futuros.

## Integracoes

- Study Engine.
- Shared.
- Design System.

## Dependencias

Nao deve possuir regras globais de prioridade fora do Study Engine.

## Exemplo de uso

Criar hooks como `useReviews` para encapsular origem de dados.
