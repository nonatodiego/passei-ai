# Goals Module

## Objetivo

Gerenciar metas de estudo.

## Responsabilidades

- Metas semanais.
- Progresso.
- Indicadores de consistencia.

## Fluxo

Services e mocks alimentam hooks; hooks alimentam components e pages.

## Integracoes

- Study Engine para impacto das metas no plano diario.
- Shared.
- Design System.

## Dependencias

Nao deve duplicar calculos de preparacao do Study Engine.

## Exemplo de uso

Expor metas por hook do modulo para manter UI desacoplada da fonte de dados.
