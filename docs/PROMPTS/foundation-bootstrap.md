Leia integralmente:

- docs/PRODUCT.md

Depois:

1. Inspecione o repositório.
2. Mostre a árvore atual.
3. Informe o que já existe.
4. Preserve arquivos válidos.
5. Crie a branch `feature/frontend-foundation`.
6. Implemente a fundação do frontend com React, TypeScript, Vite, Tailwind, React Router, Recharts e Lucide React.
7. Não implemente login.
8. Use tema light.
9. Crie layout, sidebar, header, rotas e páginas iniciais.
10. Use dados mockados.
11. Execute lint, build e testes.
12. Corrija erros.
13. Atualize ROADMAP e CHANGELOG.
14. Faça commit convencional.
15. Apresente resumo completo.
