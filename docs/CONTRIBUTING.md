## Como contribuir

1. Crie ou escolha uma issue.
2. Crie uma branch.
3. Implemente apenas o escopo.
4. Adicione testes.
5. Atualize documentação.
6. Execute lint e build.
7. Faça commit convencional.
8. Abra Pull Request.

## Pull Request

Inclua:

- objetivo;
- mudanças;
- screenshots;
- testes;
- riscos;
- documentação atualizada.

## Código de conduta

- Respeito.
- Clareza.
- Colaboração.
- Críticas técnicas sem ataques pessoais.
