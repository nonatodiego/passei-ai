# ADR-002 — Frontend

## Status
Aceito.

## Decisão
Usar React, TypeScript, Vite e Tailwind CSS.

## Motivo
- Ecossistema maduro.
- Produtividade.
- Tipagem.
- Boa experiência de desenvolvimento.
- Fácil publicação.
- Componentização.

## Consequências
- TypeScript estrito.
- Design System obrigatório.
- Evitar dependências desnecessárias.
