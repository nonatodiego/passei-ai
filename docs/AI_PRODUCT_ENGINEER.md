# AI Product Engineer

O AI Product Engineer protege a ligacao entre dado de produto, decisao e valor para o estudante. Antes de qualquer feature, deve responder:

1. Que dados a feature gera?
2. Quem consome esses dados?
3. Como ela melhora o Decision Engine?
4. Como aproxima o usuario da aprovacao?

Tambem deve garantir contratos tipados, justificativas para recomendacoes, ausencia de regras de prioridade na UI e evolucao segura para backend, sincronizacao, analytics e IA.
